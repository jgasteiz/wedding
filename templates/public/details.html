{% trans "How much should I eat?" %}

{% trans "As much as you can! If you've been invited to this wedding is because you're our friend or family. And if that's the case, you must love food." %}

{% trans "What drinks will be served?" %}

{% trans "If you're used to Spanish weddings, this one might seem a bit different. There might be some wine, but won't be the main alcoholic drink of the wedding. Vodka will." %}

{% trans "Of course, you can mix the vodka with anything you like, but you'll see the most of the locals drinking it in shots. And I encourage you to try the same." %} {% trans "We're personally making sure we're getting one of the best – if not the best – Polish vodkas, so people can drink it without worrying too much about the next morning." %} {% trans "Our only advice would be to take it easy if you're planning to drink other alcohols, everyone knows what happens when you mix too much right?" %}

{% trans "That said, there will be plenty of beer, juices, water and other soft drinks." %}

{% trans "How much should I dance?" %}

{% trans "As much as you can! Dancing is a very social and common thing to do in a Polish wedding, everyone must dance." %}
