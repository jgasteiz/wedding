{% trans "Friday 24th June" %}

{% blocktrans %} You should arrive Przeworsk at some point during Friday and get a proper night sleep before Saturday. You will need it. {% endblocktrans %}

{% trans "Saturday 25th June" %}

{% trans "Morning, before 14:00" %}

{% blocktrans %} Do you need hairdressing / make-uping / nail-painting / tie-tying? Now it's the time. {% endblocktrans %}

{% blocktrans %} For your own good, whatever you do this morning, don't get drunk. {% endblocktrans %}

{% trans "14:00" %}

{% blocktrans %} Get to the church! The ceremony will begin at 14:00 and it will last around 1 hour. {% endblocktrans %}

{% trans "15:00-15:30" %}

{% blocktrans %} A big bus and several cars will start going to the wedding venue, Ranczo, in Łopuszka Mała. {% endblocktrans %}

{% trans "15:30-16:30" %}

{% blocktrans %} Wedding reception. Magda and Javi will get to the venue, drink the inaugural shots of vodka, bread and salt. {% endblocktrans %}

{% blocktrans %} If there are any presents, they should be given now. {% endblocktrans %}

{% trans "16:30-00:00 Main feast" %}

{% trans "Eat, Dance, Drink, Repeat (5 times)" %}

{% blocktrans %}The party starts with a traditional chicken soup and it's followed by a main meat dish. After that, you should expect the First Dance and straight after the vodka will start flowing. Then, you should expect another meat/main dish every couple of hours and lots of dancing between food and drinks.{% endblocktrans %}

{% blocktrans %}Additionally, there will be a Polish "village table" with other popular Polish dishes such as pierogi, kiełbasa, smoked cheese, fish and a selection of moonshine – careful here, this vodka is usually stronger than the regular one. Most likely, there will be also a "Spanish table" with stuff such as chorizo, queso, salchichón, lomo and more.{% endblocktrans %}

{% blocktrans %}If you are vegetarian/vegan or have some kind of food intolerance, please get in touch with us and we'll try to sort something out{% endblocktrans %}: {% trans "Contact" %}

{% trans "What kind of drinks will be served?" %}

{% blocktrans %}Mainly vodka. We've already got over 100 bottles of Pan Tadeusz{% endblocktrans %}

{% blocktrans %}There will be a big selection of soft drinks that go well with vodka. Some people don't like straight vodka and prefer to mix it with coke or juices – it's perfectly fine.{% endblocktrans %}

{% blocktrans %}For the people who don't like vodka at all, there will be also plenty of beer.{% endblocktrans %}

{% trans "Dance, dance, dance" %}

{% blocktrans %}As much as you can! Dancing is a very social and common thing to do in a Polish wedding, everyone must dance.{% endblocktrans %}

{% trans "Some random invitee is insisting to drink with him, what should I do?" %}

{% blocktrans %}You won't offend (too much) anyone by not drinking when you're offered a toast, but it's always better if you accept. A good trick is to only drink half shots when you feel you've already drunk enough.{% endblocktrans %}

{% trans "00:00" %}

{% blocktrans %} There will be some activities and games for the singles. I promise they are fun. The last dish will be served after that, probably a beetroot soup with dumplings. {% endblocktrans %}

{% trans "01:00-N After party" %}

{% blocktrans %} Some people will start leaving after midnight, but that doesn't mean the party is over! The music band will probably stay until 2 and we'll have some backup music to play until 4-5-6. Buses and other transports will start coming and going between the venue and Przeworsk hotels. {% endblocktrans %}
